### 法规引用规范

#### 标准引用格式

```markdown
# 完整引用格式
**《[法规全称]》**（[年份] [修正/修订/通过]）第[条款号]条

# 示例
根据**《中华人民共和国文物保护法》**（2024修正）第十条规定...

**《历史文化名城名镇名村保护条例》**（2008）第六条要求...

**《中华人民共和国审计法》**（2021修正）第二条规定...
```

#### 条款内容引用

```markdown
# 直接引用条款内容
**《中华人民共和国文物保护法》**（2024修正）第二条规定：
"文物，是指人类在历史发展过程中遗留下来的具有历史、艺术、科学价值的遗迹和遗物。"

# 转述引用
根据**《中华人民共和国审计法》**（2021修正）第二条规定，
审计机关对"国务院各部门和地方各级人民政府及其各部门的财政收支"
进行审计监督。
```

#### 多条款引用

```markdown
# 列举多个条款
**《中华人民共和国文物保护法》**（2024修正）第十条、第十一条、第十二条规定：
- 第十条：[内容]
- 第十一条：[内容]
- 第十二条：[内容]
```

## 审计准则体系

### 中国审计准则体系

#### 1. 注册会计师审计准则（CAS）

**制定机构**：财政部
**最新版本**：2019年修订，2021年部分修正

**核心准则**：
- **基本准则**：CAS No. 1101 - 1200 系列
- **具体准则**：CAS No. 1201 - 1600 系列
- **实务指南**：CAS No. 1601 - 1800 系列

**引用格式**：
```markdown
**《中国注册会计师审计准则第[准则号]号——[准则名称]》**（[年份]）
```

#### 2. 政府审计准则

**制定机构**：审计署
**最新版本**：2010年发布，2021年修订

**核心准则**：
- **基本准则**：政府审计准则第1号
- **通用审计准则**：第2-10号
- **专业审计准则**：第11-30号

**引用格式**：
```markdown
**《政府审计准则第[准则号]号——[准则名称]》**（[年份]）
```

#### 3. 内部审计准则

**制定机构**：中国内部审计协会
**最新版本**：2013年发布，2020年修订

**核心准则**：
- **基本准则**：内部审计准则第1101号
- **具体准则**：第1201-1400号系列
- **实务指南**：第2101-2300号系列

**引用格式**：
```markdown
**《内部审计准则第[准则号]号——[准则名称]》**（[年份]）
```

### 国际审计准则

#### ISA（国际审计准则）

**制定机构**：国际审计与鉴证准则理事会（IAASB）
**适用范围**：国际通用，中国准则与之趋同

**引用格式**：
```markdown
**《国际审计准则第[ISA号]号（ISA [数字]）——[准则名称]》**
```

## 审计专业术语

### 核心术语对照表

| 中文术语 | 英文术语 | 定义 | 使用场景 |
|---------|---------|------|---------|
| 审计 | Audit | 独立检查、评价和鉴证活动 | 通用 |
| 审计证据 | Audit Evidence | 审计人员获取的证明材料 | 审计程序 |
| 重要性 | Materiality | 错报或漏报的严重程度 | 审计计划 |
| 审计风险 | Audit Risk | 审计意见不当的风险 | 风险评估 |
| 内部控制 | Internal Control | 组织内部的管理控制体系 | 控制测试 |
| 实质性程序 | Substantive Procedures | 直接检查交易和余额 | 审计实施 |
| 分析程序 | Analytical Procedures | 分析财务数据关系 | 审计方法 |
| 审计工作底稿 | Audit Working Papers | 审计过程记录 | 审计文档 |
| 审计报告 | Audit Report | 审计意见书面文件 | 审计结果 |
| 审计意见 | Audit Opinion | 审计结论 | 审计报告 |

### 文化遗产审计专用术语

| 中文术语 | 定义 | 审计关注点 |
|---------|------|-----------|
| 文化遗产 | 具有历史、艺术、科学价值的文化遗存 | 保护责任、保护措施 |
| 文物 | 人类历史发展过程中遗留下的遗迹和遗物 | 文物安全、文物管理 |
| 非物质文化遗产 | 各种社会实践、观念表达、表现形式 | 传承保护、传承人 |
| 文化遗产保护 | 防止文化遗产破坏、损失的措施 | 保护投入、保护效果 |
| 文化遗产传承 | 确保文化遗产延续、发展的活动 | 传承活动、传承效果 |
| 不可移动文物 | 不能移动的文物（古文化遗址、古墓葬等） | 修缮工程、保护规划 |
| 可移动文物 | 可以移动的文物（艺术品、实物等） | 藏品管理、展览管理 |
| 文物保护单位 | 具有价值的文物点 | 保护措施、保护资金 |
| 传承人 | 掌握非遗技艺的个人 | 传承活动、补助资金 |

## 数据验证优先级

### 优先级1：法律法规库（最高）

```bash
# 使用 Grep 查询法规库
Grep -pattern "关键词" -path="references/laws-database/markdown文件\*.md"
```

**适用内容**：

```bash

```

**适用内容**：
- 统计数据查询

### 优先级3：标注待核实（保底）

**适用内容**：
- 暂时无法查询到的数据
- 需要人工核实的复杂内容
- 缺少来源的历史数据

**标注格式**：
```markdown
**[待核实：[说明需要核实的内容]]**
```

## 禁止行为清单

### 严禁编造的内容

1. ❌ 法律法规条款内容
2. ❌ 统计数据（数字、百分比、排名）
3. ❌ 案例事实（公司名、人名、时间、金额）
4. ❌ 审计准则内容
5. ❌ 学术引用（作者、年份、页码）

### 违规后果

- 发现编造 → 整章节重新审核和修正
- 引用不完整 → 标注"[待核实]"并补充
- 数据无来源 → 必须提供来源或删除
