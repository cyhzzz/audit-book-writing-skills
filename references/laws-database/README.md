## 核心法规（Top 10）

### 1. 中华人民共和国文物保护法（2024修正）

- **地位**：文物保护领域基本法律

### 2. 中华人民共和国非物质文化遗产法

- **地位**：非遗保护领域基本法律
- **审计相关**：非遗保护、传承、传播
- **常用条款**：第二条（非遗定义）、第十二条（传承人）

### 3. 历史文化名城名镇名村保护条例

- **地位**：历史名城保护专门法规
- **审计相关**：保护规划、保护措施
- **常用条款**：第三条（保护原则）、第六条（保护规划）

### 4. 文物保护工程管理办法

- **地位**：文物保护工程管理规范
- **审计相关**：工程审批、工程监理、工程验收
- **常用条款**：第六条（资质要求）、第十二条（验收标准）

### 5. 博物馆管理办法

- **地位**：博物馆管理专门规章
- **审计相关**：藏品管理、展览管理、教育活动

## 法规查询方法

### 使用 Grep 工具查询

```bash
# 在 skill 知识库中查询
Grep -pattern "关键词" -path="references/laws-database/*.md"

# 示例1：查询"审计"相关法规
Grep -pattern "审计" -path="references/laws-database/*.md"

# 示例2：查询"保护"相关法规
Grep -pattern="保护" -path="references/laws-database/*.md"

# 示例3：查询"资金"相关法规
Grep -pattern="资金" -path="references/laws-database/*.md"
```

## 法规引用标准格式

### 完整引用格式

```markdown
**《[法规全称]》**（[年份] [修正/修订/通过]）第[条款号]条
```

### 引用示例

```markdown
# 示例1：单一条款引用
根据**《中华人民共和国文物保护法》**（2024修正）第十条规定，
"国家发展文物保护事业，建立健全文物保护工作协调机制。"

# 示例2：多条款引用
**《中华人民共和国审计法》**（2021修正）第二条、第三条规定：
- 第二条：[内容]
- 第三条：[内容]

# 示例3：条款内容引用
**《历史文化名城名镇名村保护条例》**（2008）第三条要求：
"历史文化名城、名镇、名村的保护应当遵循科学规划、严格保护的原则。"
```

   - 在文件中搜索条款号
   - 核实条款内容

5. **按标准格式引用**
   - 使用标准引用格式
   - 标注法规全称、年份、条款号

## 注意事项

1. **法规时效性**
   - 注意法规的最新修正时间
   - 优先使用最新版本
   - 如有疑问，查询官方渠道

2. **条款准确性**
   - 引用时必须核对原文
   - 条款号必须准确
   - 不得凭记忆引用

3. **完整引用**
   - 引用时必须注明法规全称
   - 必须注明年份和修正/修订情况
   - 必须注明具体条款号

4. **禁止编造**
   - 绝对禁止编造法规内容
   - 绝对禁止猜测条款号
   - 如不确定，标注"[待核实]"
