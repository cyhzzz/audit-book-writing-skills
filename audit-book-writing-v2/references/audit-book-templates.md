# 审计书籍写作模板

本文档包含审计书籍章节优化的完整模板，参考实际案例设计。

---

## 输出文件组织结构

```
章节名/
├── 评审记录/
│   └── 第X章-综合四轮评审.md    ⭐ 四轮评审合并记录
└── 优化输出/
    ├── 第X章-综合优化说明.md
    ├── 第X章-优化对比.md
    ├── 第X章-优化稿.md          ⭐ 核心输出
    └── 第X章-完成报告.md
```

**说明**：
- 四轮评审合并为一个文件，便于整体查看
- 文件命名格式：`第X章-xxx.md` 或 `[章节名]-xxx.md`
- 评审记录和优化输出分开存放，便于管理

## 四轮评审标准（审计书籍专用）

### 输出文件：`第X章-综合四轮评审.md`

### 审计书籍四轮评审要点

#### 第1轮：内容准确性审查

**审计领域特定检查项**：

1. **法规引用准确性**
   - 法规名称是否准确
   - 条款号是否正确
   - 发布日期、生效日期是否完整
   - 引用格式是否规范

2. **概念定义准确性**
   - 审计术语是否准确
   - 文化遗产概念是否准确
   - 专业术语使用是否统一

3. **数据支撑准确性**
   - 统计数据是否有来源
   - 案例数据是否准确
   - 资金数额是否准确

4. **事实陈述准确性**
   - 审计案例事实是否准确
   - 审计程序描述是否准确
   - 审计结论是否合理

#### 第2轮：逻辑连贯性审查

**审计领域特定检查项**：

1. **章节逻辑连贯性**
   - 审计逻辑是否清晰
   - 程序描述是否连贯
   - 结论推导是否合理

2. **概念层次连贯性**
   - 概念体系是否完整
   - 概念关系是否清晰
   - 层次结构是否合理

3. **论证逻辑性**
   - 审计证据是否充分
   - 论证链条是否完整
   - 结论是否自然导出

#### 第3轮：适用性审查

**审计领域特定检查项**：

1. **审计指引实用性**（P0级，800-1500字）
   - 范畴界定标准
   - 审计重点内容
   - 审计方法步骤
   - 常见问题处理

2. **框架实用性**
   - 概念框架表格
   - 分类对照表格
   - 标准对比表格
   - 问题-法规对照表

3. **案例适用性**
   - 典型案例说明
   - 审计过程描述
   - 审计发现总结
   - **审计启示**（800-1500字）

#### 第4轮：格式规范化审查

**审计领域特定检查项**：

1. **标题层级规范**
   - 一级标题：第X章
   - 二级标题：一、二、三
   - 三级标题：（一）（二）（三）
   - 四级标题：1. 2. 3.

2. **法规引用格式**
   - 法规全称 + 年份 + 条款号
   - 统一格式，完整引用

3. **术语使用一致性**
   - 审计术语统一
   - 文化遗产术语统一
   - 简称使用恰当

## 法规引用标准格式

### 单一法规引用

```markdown
# 标准格式
**《[法规全称]》**（[年份] [修正/修订/通过]）第[条款号]条

# 示例
根据**《中华人民共和国文物保护法》**（2024修正）第十条规定，
"国家发展文物保护事业，建立健全文物保护工作协调机制。"
```

### 多条款引用

```markdown
# 并列引用
```
