# Audit Book Writing Pro v2.0

**审计书籍撰写与评审增强版**

## 版本特点

v2.0是审计书籍撰写系统的增强版本，在v1.0基础上增加了以下核心功能：

- ✅ **四轮评审系统** - 内容准确性、逻辑连贯性、适用性、格式规范化
- ✅ **双模式优化** - 修订模式（95%-120%字数控制）和重构模式（大幅调整）
- ✅ **多文件留痕输出** - 5个评审文件 + 1个优化稿
- ✅ **100个法律法规知识库** - 覆盖国家/省/市三级法规
- ✅ **审计应用指引模板** - 800-1500字标准模板
- ✅ **审计案例启示模板** - 800-1500字标准模板
- ✅ **字数严格控制** - 修订模式下字数必须控制在原稿的95%-120%

## 适用场景

- 审计书籍章节优化（文化遗产审计、财务审计等）
- 审计实务指南编写
- 案例分析与解读
- 法规解读与应用
- 需要多文件留痕和严格质量控制的审计写作

## 核心原则

### ⚠️ 数据验证原则（最高优先级）

**绝对禁止编造任何内容、数据或引用材料**

#### 数据验证优先级

**优先级1（最高）**: 法律法规库 → 使用Grep在法规库中查询
**优先级2（中等）**: 联网查询 → 使用WebSearch查询权威网站
**优先级3（保底）**: 标注待核实 → 明确标注需要人工核对

### ⚠️ 字数控制原则（严格标准）

**修订模式**: 优化稿字符数必须控制在原稿的 **95%-120%** 区间内

```
原稿字符数 = X
优化稿字符数 = Y
比例 = Y / X × 100%

合格标准：95% ≤ 比例 ≤ 120%
```

### 模式选择原则

- **修订模式**: 适用于原稿有明确框架和核心观点（保留原稿核心观点和内涵）
- **重构模式**: 适用于原稿结构混乱或观点不明确（必须先确认观点和结构）

## 快速开始

### 在Claude Code中使用

```bash
# 安装v2.0版本
/plugin install audit-book-writing-v2@cyhzzz/audit-book-writing-skills
```

### 使用示例 - 修订模式

```
请使用 audit-book-writing-v2 技能，使用修订模式优化以下章节：

# 第3章 审计的基本概念

[章节内容...]

请执行四轮评审，生成5个评审文件 + 1个优化稿。
字数请控制在原稿的95%-120%之间。
```

### 使用示例 - 重构模式

```
请使用 audit-book-writing-v2 技能，使用重构模式优化以下章节：

# 第5章 内部控制审计

[章节内容...]

请先提取学术观点并确认，然后提出结构优化方案。
确认后执行四轮评审并生成重构后的优化稿。
```

## 输出文件说明

### 评审文件（5个）

1. **第1轮评审_内容准确性.md** - 检查法规引用准确性、案例准确性、术语准确性
2. **第2轮评审_逻辑连贯性.md** - 检查章节逻辑、论证逻辑、一致性
3. **第3轮评审_适用性.md** - 检查程序可行性、案例代表性、建议价值
4. **第4轮评审_格式规范化.md** - 检查引用格式、表格一致性、标题层级
5. **四轮评审汇总.md** - 汇总所有评审意见和改进建议

### 优化稿（1个）

**优化稿_章节名称.md** - 基于评审意见优化后的最终版本
