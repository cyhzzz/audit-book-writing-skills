# 审计书籍智能校对优化技能 (Audit Book Smart Proofreading)

**版本:** v4.0.0
**开发者:** OpenClaw 🦞
**适用模型:** GLM-4.7 / GLM-4.6 / Claude 3.5 Sonnet

## 📖 简介

审计书籍智能校对优化技能是审计书籍编写项目的 v4 版本，**重点优化校对优化场景**。基于真实审计书籍章节的 26 步校对优化流程，提炼出智能化、系统化的文档校对优化方法。

### 核心价值

✅ **法规准确** - 多源交叉验证，确保法规表述 100% 准确
✅ **案例权威** - 强制使用官方网站案例，标注来源
✅ **内容合规** - 严格围绕主题，无离题内容
✅ **效率提升** - 自动化处理，效率提升 3-5 倍
✅ **质量保证** - 多维度检查，零失误目标

### 适用场景

- 审计书籍章节校对优化
- 法规类文档审核
- 案例型文档标准化
- 审计报告质量提升
- 文档法规合规性检查

## 🎯 核心功能

### 1. 智能标记系统

自动识别并标记文档中 8 种类型的内容：

| 标记 | 类型 | 说明 |
|------|------|------|
| FA1 | 法规条文 | 直接属于法律法规条文的内容 |
| FA2 | 法规延伸 | 与法规条文相关的延伸内容 |
| FB1 | 条款引用 | 含有"第××条"形式的内容 |
| FB2 | 条款延伸 | 与条款引用相关的延伸内容 |
| FC1 | 发文字号 | 含有发文字号的内容 |
| FC2 | 文号延伸 | 与发文字号相关的延伸内容 |
| A1 | 显式地名案例 | 含有具体地名的案例 |
| A2 | 隐式地名案例 | 含有隐含地名的案例 |

### 2. 多源法规验证

支持 1-8 个法规汇总文件交叉验证：
- 逐一比对每个标记的法规条文
- 确保表述完全一致
- 修正条款序号和文字表述
- 标记无法验证的内容并清理

### 3. 权威案例替换

从官方网站查找并替换权威案例：
- 优先级：国家文物局 > 浙江省文物局 > 其他省份文物局
- 标注案例来源完整网址
- 无合适案例自动删除
- 严禁联想编造

### 4. 内容智能扩充

基于法规自动补充相关内容：
- 法规条文原文
- 相关案例
- 审计应用

### 5. 结构优化整理

自动归并重复内容，优化文档结构：
- 识别并归并"审计内容"与"审计重点"的重复内容
- 消除逻辑冲突
- 重新分配内容，保持结构清晰

## 🚀 快速开始

### 前置条件

**必传文件:**

1. **待校对文档** - Word 格式 (.doc / .docx)
2. **法规汇总文件** - Word 格式 (.docx)，至少 1 个

**可选配置:**

```yaml
审计主题:
  名称: "古人类遗址保护责任审计"
  核心要求: "围绕古人类遗址保护，不涉及资金使用"

文档结构要求:
  最小字数: 5000字
  保持原有结构: true
```

### 使用流程

#### 第一步：初始化配置

提供待校对文档路径、法规汇总文件路径、审计主题等信息。

#### 第二步：智能标记

系统自动识别并标记文档中 8 种类型的内容。

#### 第三步：多源法规验证

与法规汇总文件交叉验证，确保法规表述准确。

#### 第四步：权威案例替换

从官方网站查找并替换权威案例。

#### 第五步：内容智能扩充

基于法规自动补充相关内容。

#### 第六步：结构优化整理

归并重复内容，优化文档结构。

#### 第七步：生成最终报告

生成校对优化完成报告和详细分阶段报告。

## 🔧 配置参数

```yaml
skill_config:
  version: "4.0.0"
  max_file_size: 50MB
  supported_formats: [".doc", ".docx"]
  default_min_word_count: 5000

marking_rules:
  enable_FA1: true
  enable_FA2: true
  enable_FB1: true
  enable_FB2: true
  enable_FC1: true
  enable_FC2: true
  enable_A1: true
  enable_A2: true

verification_rules:
  max_regulation_files: 8
  similarity_threshold: 0.7
  strict_mode: true

case_replacement_rules:
  priority_sites:
    - "国家文物局官方网站"
    - "浙江省文物局官方网站"
    - "其他各省文物局官方网站"
  allow_case_deletion: true

content_expansion_rules:
  max_expansion_per_regulation: 3
  include_audit_application: true
  forbid_off_topic: true

structure_optimization_rules:
  merge_duplicate_content: true
  resolve_conflicts: true
  preserve_structure: true
```

## 🤝 使用建议

1. **确保法规汇总文件完整** - 建议提供 8 个法规汇总文件以获得最佳验证效果
